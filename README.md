# VIT Full Stack API Challenge

## Overview
REST API that processes arrays and returns categorized data including numbers, alphabets, and special characters.

## API Endpoint
**POST** `/bfhl`

## Request Format
Send POST request with JSON body:
```json
{
  "data": ["a","1","334","4","R", "$"]
}
```
